# Md Shafi Kamal - Developer Portfolio

A modern, responsive developer portfolio website showcasing web development experience for Web Design & Development RPL Level-03 assessment.

## 🎯 Overview

This portfolio website is built using:
- **HTML5** - Semantic markup
- **CSS3** - Modern styling with animations
- **Bootstrap 5** - Responsive grid system
- **JavaScript (ES6)** - Interactive features
- **Font Awesome** - Icon library

## 📋 Features

### Sections
1. **Home** - Hero section with call-to-action buttons
2. **About** - Professional summary and key focus areas
3. **Skills** - Organized skills by category (Backend, Frontend, DevOps & Tools)
4. **Projects** - Featured projects with descriptions and tech stack
5. **Experience** - Timeline view of professional experience
6. **Contact** - Contact information and message form

### Design Features
- ✨ Clean and modern UI
- 📱 Fully responsive design (mobile, tablet, desktop)

- 🔗 Smooth scrolling navigation

- 💡 Interactive skill cards with hover effects

## 🚀 Getting Started

### Prerequisites
- A modern web browser
- A code editor (VS Code recommended)
- Git (for version control)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/shafikamal/portfolio.git
   cd portfolio
   ```

2. **Open the portfolio**
   - Simply open `index.html` in your web browser
   - Or use a local server for better testing

3. **Using Live Server (recommended)**
   ```bash
   # Install Live Server extension in VS Code
   # Right-click on index.html and select "Open with Live Server"
   ```

## 📝 Customization

### Update Personal Information
Edit the following in `index.html`:

```html
<!-- Email -->
<a href="mailto:shafi.kamal@example.com">shafi.kamal@example.com</a>

<!-- Phone -->
<a href="tel:+8801234567890">+880 1234 567890</a>
```

### Add/Modify Projects
Find the "Featured Projects" section and update the project cards:

```html
<div class="col-lg-4 col-md-6">
    <div class="project-card">
        <!-- Update project details here -->
    </div>
</div>
```

### Change Colors
Update the primary color in `styles.css`:

```css
/* Search for #0d6efd and replace with your color */
/* Also update #0dcaf0 (secondary color) */
```

### Add Social Media Links
Update the social links in the hero section:

```html
<div class="social-links mt-5">
    <a href="https://github.com/shafikamal" class="social-icon">
        <i class="fab fa-github"></i>
    </a>
    <a href="https://linkedin.com/in/shafikamal" class="social-icon">
        <i class="fab fa-linkedin"></i>
    </a>
</div>
```
